Escuela de código de PILARES El objetivo de la Escuela de código de PILARES es contribuir a la participación principalmente de las mujeres en el mundo de la programación, en particular como programadoras WEB, y su inserción en instituciones, corporaciones o bien de forma independiente. La Escuela de código es una propuesta con perspectiva de género. Actualmente la Escuela de código ofrece dos módulos: Conociendo el ambiente de computación Introducción al cómputo y a sistemas Linux Elaboración de sistemas web Conocimientos para poder elaborar páginas (estáticas y dinámicas) y aplicaciones web. Las participantes de la Escuela de código adquirirán conocimientos en: Sistemas operativos Linux Git Programación web Front-end: HTML, CSS, JavaScript, Bootstrap, DOM Back-end: Node.js, Express.js, Sails.js, PHP, Laravel, MongoDB, SQLite, REST-API, MVC.

Habilidades

Usar servicios de nube como Google Drive. Navegar de forma segura en internet. Crear páginas web estáticas y dinámicas. Crear REST-API’s que manipula en bases de datos. Utilizar REST-API’s para crear aplicaciones web. Crear formularios que almacenen la información en algún gestor de bases de datos. Utilizar Git para tener el control de las versiones de sus códigos y tenerlos respaldados en la nube.
